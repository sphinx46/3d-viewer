# Changelog
---

## [0.0.1] — 2026-01-16

### Added
- Первый бета-релиз приложения **3d-viewer**
- Поддержка загрузки и сохранения моделей в формате Wavefront `.obj`
- Экспорт/импорт сцен в JSON и собственном бинарном формате `.3dcene`
- Стандартные примитивы: куб, сфера, цилиндр, плоскость, капсула, чайник Юты
- Полнофункциональный редактор трансформаций (перемещение, вращение, масштаб)
- Система материалов с поддержкой цвета, блеска, прозрачности, отражения
- Загрузка текстур (PNG, JPG, BMP, GIF)
- Управление несколькими камерами с настройкой FOV, позиции и цели
- Древовидная панель сцены с контекстным меню
- Горячие клавиши для основных действий (см. `README.md`)
- Поддержка тем оформления (светлая/тёмная)
- Вспомогательные элементы: сетка, координатные оси
- Автоматическое кэширование недавних файлов
- Справка по горячим клавишам и документации

### Changed
- Интерфейс полностью реализован на JavaFX без сторонних 3D-библиотек
- Рендеринг осуществляется через программный растеризатор на `Canvas`

### Fixed
- Исправлена ошибка отображения нормалей при загрузке некоторых `.obj`-файлов
- Улучшена производительность при отрисовке сцен с большим количеством полигонов

### Technical
- Проект собран с помощью Maven (multi-module архитектура)
- Требуется Java 17+, но протестирован и совместим с Java 21
- Включена поддержка логирования через SLF4J + Logback
- Написаны базовые unit-тесты (JUnit)
